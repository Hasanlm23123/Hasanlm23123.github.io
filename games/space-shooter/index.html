How It Works

Gameplay Notes

  • Move the ship horizontally and avoid incoming asteroids.
  • Each asteroid you hit with a bullet resets and adds one point.
  • If an asteroid touches the ship, the run ends immediately.
  • Press Enter after a loss to restart from the keyboard.
The spaceship and asteroid art are pulled into the site so the game can run directly on GitHub Pages without a Python runtime.
